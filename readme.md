guessing_game
-------------------------

動作: プログラムは1から100の間のあるランダムな数字を生成します。 そしてその値の予想値の入力を促します。 予想値を入力すると大きすぎるあるいは小さすぎると教えてくれます。 当たったらおめでとうと言ってくれます。
